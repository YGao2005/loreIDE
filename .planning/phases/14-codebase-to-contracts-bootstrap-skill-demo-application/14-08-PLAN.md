<objective>
Add Stage 0.5 to the codebase-to-contracts pipeline: ingest historical decision artifacts (CLAUDE.md, git log, ADRs, CHANGELOG) and produce substrate nodes via the existing Phase 11 distiller — closing the pitch claim "Day 0 bootstraps from existing CLAUDE.mds, ADRs… via importers" (`PITCH.md:23,127-128`).

Today, Phase 14's bootstrap skill produces only contracts (Intent / Role / Inputs / Outputs / Side effects). The substrate side (`con-*`, `dec-*`, `open-*` nodes) populates only from live Claude Code sessions via Phase 10/11. That leaves "Day 0 substrate" empty until enough sessions accumulate — a multi-week ramp that breaks the "point at any repo and you have institutional knowledge immediately" promise.

Stage 0.5 closes that gap. Every codebase already has historical decision artifacts: a CLAUDE.md in the repo root, the last ~200 commit messages, optional `docs/adr/*.md` ADRs, optional `CHANGELOG.md` entries, optional `DECISIONS.md`. These are *exactly* the kind of unstructured prose the Phase 11 distiller already extracts intent from — the only difference is they're not Claude session JSONL. Treat each artifact as a synthetic episode, pipe through the same distiller, persist to the same `substrate_nodes` table with `provenance.source='historical'`. Output: 10–30 substrate nodes per repo at bootstrap time.

Architecture is reuse-not-rebuild:
- Existing `distiller/pipeline.rs` already does claude -p → JSON-schema-validated substrate node extraction → SQLite upsert → `substrate:nodes-added` event. We do not rewrite it.
- Stage 0.5 produces synthetic episodes that match the same payload shape; a thin Rust adapter (`distiller/historical.rs`) routes them through.
- A dedicated prompt (`prompts/distill-historical.txt`) handles the non-chat shape — commit messages need different extraction rules than dialogue.
- Cost is bounded: cap at 50 distillation calls per run (~$1.50 ceiling).

Output: Stage 0.5 script + adapter + prompt + schema + tests + SKILL.md update + smoke against bootstrap-demo-target showing ≥5 substrate nodes derived from Marginalia's existing artifacts (16 commits + README; CLAUDE.md will be synthesized into Marginalia as part of Task 0 below to give the smoke a richer surface).

This is the WAVE 4 plan — depends on 14-03 (Stage 0 preflight infra + .staging/ structure). Independent of 14-04/05; can land in parallel. Plan ordering rationale: 04 → 08 → 05 → 07 → 06.
